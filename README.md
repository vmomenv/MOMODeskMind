## 桌面桌宠AI MOMODeskMind

### 1. 项目简介
- **项目名称**：MOMODeskMind
- **项目描述**：一个基于QWEN2.5-0.5b语言模型的桌面宠物应用，提供智能交互、天气预报、日程提醒等功能。
- **目标用户**：个人用户、办公人员、学生等。

### 2. 即将支持的功能特性
-  [ ]  **智能交互**：支持自然语言对话，回答用户的问题。
-  [ ]  **天气预报**：显示当前天气和未来几天的天气预报。
-   [ ] **日程提醒**：帮助用户管理日程。
-   [ ] **个性化设置**：用户可以自定义宠物的外观和行为。

### 3. 技术栈
- **前端**：Qt6
- **后端**：C++
- **语言模型**：QWEN2.5-0.5b
- **构建工具**：CMake

### 4. 安装与运行
#### 4.1 环境准备
- **操作系统支持**：现仅支持windows系统
- **依赖软件**：Qt6 SDK（额外下载Qt6WebEngineWidgets）, CMake, GCC/G++

#### 4.2 安装步骤
1. 克隆项目仓库：
   ```sh
   git clone https://github.com/vmomenv/MOMODeskMind.git
   cd MOMODeskMind
   ```
2. 安装Qt6 SDK：
   - 下载并安装Qt6 SDK：[Qt官方网站](https://www.qt.io/download)
3. 安装依赖：
   ```sh
   cmake .
   make
   ```
